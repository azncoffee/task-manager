## Description
A task management application using React and Express

## Tech Stack
- Frontend: React, Axios
- Backend: Node.js, Express, MongoDB

## Getting Started

### Backend Setup
```bash
cd backend
npm install express mongoose cors
node server.js
```

### Frontend Setup
```bash
cd frontend
npx create-react-app .
npm install axios
npm start
```
